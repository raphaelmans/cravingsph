# US-ADMIN-004: View Platform-Wide Order Statistics

**As a** platform admin,
**I want** to see platform-wide order metrics,
**So that** I can track overall growth and activity.

## Acceptance Criteria

- **Given** the platform has order activity, **When** I view admin analytics, **Then** I can see aggregated order statistics across restaurants.
- **Given** there is no data for a selected period, **When** I view the dashboard, **Then** the metrics render as zero or empty without failing.
