# US-ADMIN-005: Manage User Accounts

**As a** platform admin,
**I want** to view or deactivate user accounts,
**So that** I can handle support and trust-and-safety cases.

## Acceptance Criteria

- **Given** platform users exist, **When** I open the admin user list, **Then** I can review user accounts and their relevant account details.
- **Given** a support or policy issue requires intervention, **When** I deactivate a user account, **Then** that account is prevented from continuing normal platform access.
