# US-ADMIN-002: List Restaurants By Verification Status

**As a** platform admin,
**I want** to view all restaurants and their verification states,
**So that** I can track onboarding progress across the platform.

## Acceptance Criteria

- **Given** restaurants exist on the platform, **When** I open the admin restaurant list, **Then** I can see each restaurant with its current verification status.
- **Given** I need to focus on a specific status, **When** I filter the list by that status, **Then** only matching restaurants remain visible.
