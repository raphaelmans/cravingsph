# US-ADMIN-001: Review Verification Submissions

**As a** platform admin,
**I want** to review restaurant verification submissions,
**So that** I can approve or reject listings before they go live.

## Acceptance Criteria

- **Given** a restaurant owner submits verification documents, **When** I open the admin review queue, **Then** I can inspect the submission details and attached documents.
- **Given** I approve or reject the submission, **When** I save the decision, **Then** the restaurant verification status is updated accordingly.
