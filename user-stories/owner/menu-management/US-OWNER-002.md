# US-OWNER-002: Add Menu Items

**As a** restaurant owner,
**I want** to create menu items with name, description, image, and base price,
**So that** customers can see what I offer.

## Acceptance Criteria

- **Given** a category exists, **When** I add a menu item with valid pricing and content, **Then** the item is saved under that category.
- **Given** required item data is missing or malformed, **When** I try to save the item, **Then** validation prevents the item from being created.
