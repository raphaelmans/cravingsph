# US-OWNER-008: Search And Filter Menu Items

**As a** restaurant owner,
**I want** to search, filter, and sort menu items in management views,
**So that** I can manage large menus efficiently.

## Acceptance Criteria

- **Given** a branch has many items, **When** I search or filter the management list, **Then** the results are narrowed to the matching menu items.
- **Given** I clear the search and filters, **When** the view refreshes, **Then** the full item set returns in the active default ordering.
