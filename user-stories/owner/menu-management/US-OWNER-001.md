# US-OWNER-001: Create Menu Categories

**As a** restaurant owner,
**I want** to create categories for a branch menu,
**So that** my menu stays organized.

## Acceptance Criteria

- **Given** I am managing a branch menu, **When** I create a category with a valid name, **Then** the category is saved for that branch with menu ordering metadata.
- **Given** category input is invalid, **When** I submit it, **Then** the system rejects the change and preserves the existing category list.
