# US-OWNER-009: Upload Menu Item Images

**As a** restaurant owner,
**I want** to upload menu item images with preview support,
**So that** my menu is visually appealing.

## Acceptance Criteria

- **Given** I am editing a menu item, **When** I upload an image, **Then** the item stores the image URL and shows a preview before save confirmation.
- **Given** an upload fails or the file is invalid, **When** I attempt the upload, **Then** the item is not updated with a broken image reference.
