# US-OWNER-007: See Daily And Weekly Summaries

**As a** restaurant owner,
**I want** to view daily and weekly order summaries,
**So that** I can track basic business volume.

## Acceptance Criteria

- **Given** my branch has order activity, **When** I view summaries, **Then** I can see aggregated order counts and totals for daily and weekly periods.
- **Given** a period has no orders, **When** I view that summary, **Then** the dashboard shows zero activity instead of failing to render.
