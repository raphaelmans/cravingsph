# US-OWNER-006: Review Full Order Details

**As a** restaurant owner,
**I want** to see the full contents of an order,
**So that** my team can prepare it correctly.

## Acceptance Criteria

- **Given** an order exists, **When** I open its detail view, **Then** I can see line items, quantities, selected modifiers, special instructions, and customer contact details.
- **Given** the order is dine-in, **When** a table number was supplied, **Then** the order detail view includes that table number for staff reference.
