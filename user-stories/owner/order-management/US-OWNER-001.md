# US-OWNER-001: View Order Dashboard Tabs

**As a** restaurant owner,
**I want** to see incoming and historical orders grouped by workflow status,
**So that** I can manage operations from one dashboard.

## Acceptance Criteria

- **Given** my branch has orders in multiple states, **When** I open the order dashboard, **Then** I can view them in Inbox, Active, Completed, and Cancelled groupings.
- **Given** a tab has no orders, **When** I open it, **Then** I see an empty state instead of stale orders from another tab.
