# US-OWNER-002: Receive New Order Notifications

**As a** restaurant owner,
**I want** to receive push notifications when new orders arrive,
**So that** I can respond quickly to incoming demand.

## Acceptance Criteria

- **Given** notifications are enabled for my account, **When** a new order is created for my branch, **Then** I receive a new-order notification.
- **Given** notifications are disabled or unavailable, **When** a new order arrives, **Then** the order still appears in the dashboard inbox.
