# US-OWNER-003: Review Submitted Payment Proof

**As a** restaurant owner,
**I want** to review payment proof submitted by customers,
**So that** I can verify payments before confirmation.

## Acceptance Criteria

- **Given** a customer submits payment proof, **When** I open the order payment details, **Then** I can see the reference number and any uploaded screenshot.
- **Given** no screenshot was uploaded, **When** I review the proof, **Then** the reference number alone is still visible for manual verification.
