# US-OWNER-002: Set A Default Payment Method

**As a** restaurant owner,
**I want** to add multiple payment methods and choose one as default,
**So that** customers see my preferred payment method first.

## Acceptance Criteria

- **Given** my organization has multiple active payment methods, **When** I mark one as default, **Then** only that method is designated as the preferred option.
- **Given** I change the default method, **When** the update is saved, **Then** customer checkout shows the new default first.
