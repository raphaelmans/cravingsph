# US-OWNER-002: Configure Acceptance Mode Per Branch

**As a** restaurant owner,
**I want** to choose manual or auto-accept mode per branch,
**So that** each branch can follow the workflow that suits its operations.

## Acceptance Criteria

- **Given** I am editing branch settings, **When** I switch the branch between manual and auto-accept, **Then** the selected acceptance mode is saved for that branch.
- **Given** I manage multiple branches, **When** I change one branch's acceptance mode, **Then** other branches keep their own independent setting.
