# US-OWNER-001: Set Branch Operating Hours

**As a** restaurant owner,
**I want** to define operating hours per branch,
**So that** customers know when a branch is open.

## Acceptance Criteria

- **Given** a branch exists, **When** I configure its weekly operating hours, **Then** those hours are saved for that branch.
- **Given** a branch is currently outside operating hours, **When** customers view or order from that branch, **Then** the app can reflect that it is closed.
