# US-OWNER-005: Toggle Online Ordering Per Branch

**As a** restaurant owner,
**I want** to turn online ordering on or off per branch,
**So that** I can temporarily pause incoming orders when needed.

## Acceptance Criteria

- **Given** a branch is actively receiving orders, **When** I turn off online ordering, **Then** new public ordering is blocked for that branch.
- **Given** I re-enable online ordering, **When** the branch returns online, **Then** customers can resume placing new orders for that branch.
