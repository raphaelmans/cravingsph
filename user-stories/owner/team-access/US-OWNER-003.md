# US-OWNER-003: Allow Managers To Operate Orders And Menu

**As a** restaurant owner,
**I want** managers to handle operational work,
**So that** I can delegate day-to-day restaurant activity.

## Acceptance Criteria

- **Given** a team member has the manager role, **When** they access the owner portal, **Then** they can manage orders, update statuses, and edit the menu within their permission scope.
- **Given** a manager attempts an action outside granted permissions, **When** access is checked, **Then** the action is denied.
