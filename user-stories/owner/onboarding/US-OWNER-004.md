# US-OWNER-004: Add Multiple Branches

**As a** restaurant owner,
**I want** to create more than one branch for my restaurant,
**So that** each physical location can have its own URL and operations.

## Acceptance Criteria

- **Given** my restaurant exists, **When** I add a branch with its own details, **Then** that branch is created under the selected restaurant.
- **Given** I have already created one branch, **When** I add another, **Then** both branches remain separately manageable with distinct slugs and settings.
