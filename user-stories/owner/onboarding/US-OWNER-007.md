# US-OWNER-007: Use Onboarding Hub View

**As a** restaurant owner,
**I want** a non-linear onboarding hub view,
**So that** I can jump directly to any incomplete setup task.

## Acceptance Criteria

- **Given** onboarding is in progress, **When** I open the hub view, **Then** I can see all setup tasks and their completion states in one place.
- **Given** I select a specific task from the hub, **When** navigation occurs, **Then** I am taken directly to that setup task without replaying the full wizard.
