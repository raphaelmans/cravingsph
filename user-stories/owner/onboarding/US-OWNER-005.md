# US-OWNER-005: Submit Verification Documents

**As a** restaurant owner,
**I want** to upload verification documents for admin review,
**So that** my restaurant can be approved to go live.

## Acceptance Criteria

- **Given** onboarding reaches verification, **When** I upload the required proof-of-ownership documents and submit, **Then** the verification request is saved for admin review.
- **Given** required verification inputs are missing, **When** I try to submit the request, **Then** the system blocks submission and identifies the missing documents.
