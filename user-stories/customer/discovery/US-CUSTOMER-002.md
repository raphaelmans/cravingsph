# US-CUSTOMER-002: Search Restaurants By Name

**As a** customer,
**I want** to search restaurants by name,
**So that** I can quickly find a specific place.

## Acceptance Criteria

- **Given** restaurants are searchable, **When** I type a restaurant name, **Then** the search results narrow to matching restaurant records.
- **Given** no restaurant matches my query, **When** search completes, **Then** I see an empty-state message instead of unrelated results.
