# US-CUSTOMER-003: Filter Restaurants By Cuisine And Location

**As a** customer,
**I want** to filter restaurants by cuisine type and city or province,
**So that** I can narrow the list to places that fit my needs.

## Acceptance Criteria

- **Given** cuisine and location filters are available, **When** I apply one or more filters, **Then** the restaurant list refreshes to only matching restaurants.
- **Given** I clear the filters, **When** the filter state resets, **Then** the full browse list returns without stale constraints.
