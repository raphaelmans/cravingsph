# US-CUSTOMER-004: Review Restaurant Cards

**As a** customer,
**I want** to see restaurant cards with useful preview information,
**So that** I can decide which menu to open.

## Acceptance Criteria

- **Given** restaurants are listed in discovery, **When** cards render, **Then** each card shows cover photo, name, cuisine tags, and a preview of popular items when available.
- **Given** some preview fields are missing, **When** the card renders, **Then** the listing still shows a usable fallback layout without breaking the grid.
