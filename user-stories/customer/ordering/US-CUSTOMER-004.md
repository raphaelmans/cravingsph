# US-CUSTOMER-004: Provide Table Number For Dine-In

**As a** customer,
**I want** to enter a table number when dine-in service requires it,
**So that** staff knows where to deliver my food.

## Acceptance Criteria

- **Given** a branch requires table numbers for dine-in, **When** I choose dine-in, **Then** the checkout form requires a table number before order submission.
- **Given** a branch does not require table numbers, **When** I choose dine-in, **Then** the table number field is optional or hidden.
