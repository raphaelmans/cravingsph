# US-CUSTOMER-001: Place Dine-In Order As Guest

**As a** customer,
**I want** to place a dine-in order without creating an account,
**So that** checkout stays frictionless in-store.

## Acceptance Criteria

- **Given** my cart is valid and dine-in is available, **When** I check out as a guest, **Then** I can submit the order without signing up.
- **Given** required dine-in details are missing, **When** I submit the order, **Then** the order is blocked until those details are provided.
