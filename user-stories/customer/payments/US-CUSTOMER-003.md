# US-CUSTOMER-003: Submit Payment Proof

**As a** customer,
**I want** to upload payment proof after paying externally,
**So that** the restaurant can verify my payment.

## Acceptance Criteria

- **Given** an order requires manual payment verification, **When** I provide a reference number and submit proof, **Then** the order stores the payment proof for restaurant review.
- **Given** the screenshot is optional, **When** I submit only a valid reference number, **Then** the payment proof is still accepted.
