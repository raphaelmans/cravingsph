# US-CUSTOMER-001: Open Menu From QR

**As a** customer,
**I want** to scan a branch QR code and open the menu immediately,
**So that** I can browse without downloading an app.

## Acceptance Criteria

- **Given** a valid branch QR code, **When** I open it on my phone, **Then** the branch menu page loads without requiring sign-in.
- **Given** a QR code for an inactive or unknown branch, **When** I open it, **Then** I see an unavailable state instead of an orderable menu.
