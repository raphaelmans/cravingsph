# US-CUSTOMER-006: Jump Between Categories

**As a** customer,
**I want** to use sticky category navigation while scrolling the menu,
**So that** I can move between sections without returning to the top.

## Acceptance Criteria

- **Given** a menu has multiple categories, **When** I scroll down the page, **Then** the category navigation remains accessible in a sticky header.
- **Given** I tap a category in the sticky header, **When** navigation runs, **Then** the page scrolls to the matching category section and highlights the active section.
