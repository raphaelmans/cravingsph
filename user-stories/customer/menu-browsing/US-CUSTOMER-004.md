# US-CUSTOMER-004: View Item Details

**As a** customer,
**I want** to see an item's details before ordering,
**So that** I can make an informed choice.

## Acceptance Criteria

- **Given** a menu item is available, **When** I open its detail view, **Then** I see its name, description, image, and current price information.
- **Given** an item has missing optional data such as description or image, **When** I open it, **Then** the detail view still renders without broken layout or missing price information.
