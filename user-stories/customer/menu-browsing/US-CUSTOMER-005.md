# US-CUSTOMER-005: View Restaurant Details

**As a** customer,
**I want** to see the restaurant's profile details while browsing,
**So that** I know where I am ordering from.

## Acceptance Criteria

- **Given** a restaurant profile is configured, **When** I view the menu page, **Then** I can see the restaurant name, cover photo, profile image, address, and contact information.
- **Given** some restaurant profile fields are not configured, **When** I open the menu page, **Then** the available information is shown and missing fields do not break the page.
