# US-CUSTOMER-002: Browse Items By Category

**As a** customer,
**I want** to browse menu items grouped by category,
**So that** I can find what I want quickly.

## Acceptance Criteria

- **Given** a branch has menu categories with available items, **When** I view the menu, **Then** items are shown under their assigned categories in menu order.
- **Given** a category has no visible items, **When** I browse the menu, **Then** the UI hides or clearly marks that empty category so navigation stays useful.
