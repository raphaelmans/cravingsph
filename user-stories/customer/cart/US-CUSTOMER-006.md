# US-CUSTOMER-006: See Cart Count In Header

**As a** customer,
**I want** to see my cart count in the header,
**So that** I always know how many items I have added.

## Acceptance Criteria

- **Given** my cart contains items, **When** I browse the menu, **Then** the header cart icon shows the current total item count.
- **Given** the cart becomes empty, **When** the last line is removed, **Then** the header count resets and the empty state is reflected immediately.
