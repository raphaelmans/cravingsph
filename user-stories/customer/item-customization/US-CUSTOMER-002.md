# US-CUSTOMER-002: Add Optional Modifiers

**As a** customer,
**I want** to add optional modifiers to an item,
**So that** I can customize my order to my preference.

## Acceptance Criteria

- **Given** an item has optional modifier groups, **When** I select or deselect modifiers, **Then** the chosen modifiers are stored on the draft order item.
- **Given** a modifier has an added cost, **When** I select it, **Then** the added price is included in the item total preview.
