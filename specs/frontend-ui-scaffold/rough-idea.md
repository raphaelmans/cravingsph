# Rough Idea: Frontend UI Scaffold

Create all frontend components and page wirings for CravingsPH using shadcn/shadcn-ui, covering:

1. **Customer portal** — Menu browsing (rebuild from legacy), cart, ordering, payments, discovery, order tracking
2. **Restaurant owner portal** — Onboarding wizard, menu management, order dashboard, payment config, settings
3. **Admin portal** — Verification review, restaurant management, platform stats

Must respect the legacy customer flow (category nav, modifier system, smart cart, bottom sheets, pill shapes, #f86006 orange brand) while filling all gaps identified in the legacy analysis (ordering, payments, discovery, onboarding, order management).
